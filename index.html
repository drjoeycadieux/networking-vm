Getting Started: Quick Setup

1

Download and Import

Download the `.ova` file and use your virtualization platform's 'Import Appliance' function to load the VM.

2

Default Credentials

The default user is networking-vm. The password is admin1234.

3

Network Configuration

The VM is set to use **NAT** by default. For network auditing or sniffing, switch the adapter to **Bridged Adapter** mode in your virtualization settings.
