Getting Started: Quick Setup

1

Download and Import

Download the `.ova` file and use your virtualization platform's 'Import Appliance' or 'File > Import' function to load the VM.

2

Default Credentials

The default user is networking-vm. The password is admin1234.

3

Network Configuration

The VM defaults to **NAT** networking. For deep network auditing or packet sniffing, switch the adapter to **Bridged Adapter** mode in your virtualization software settings for full network access.
